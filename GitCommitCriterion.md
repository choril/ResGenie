# 使用 Conventional Commits 规范
[https://www.conventionalcommits.org/](https://www.conventionalcommits.org/)

### 类型：
- feat:     新功能
- fix:      修复bug
- docs:     文档更新
- style:    代码格式（不影响功能）
- refactor: 重构代码
- test:     测试相关
- chore:    构建过程或工具变更

# 示例：
```code
git commit -m "feat: 添加学术论文检索智能体"
git commit -m "fix: 修复 FastAPI 路由注册问题"
git commit -m "docs: 更新 API 使用文档"
git commit -m "chore: 更新 pyproject.toml 依赖版本"
```
